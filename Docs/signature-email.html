Instructions Outlook :
1. Ouvrir ce fichier dans un navigateur (double-clic)
2. Selectionner tout le bloc signature (Ctrl+A dans le bloc blanc)
3. Copier (Ctrl+C)
4. Dans Outlook : Parametres (roue dentee) > Courrier > Composer et repondre > Signature
5. Coller (Ctrl+V) dans le champ signature
6. Enregistrer
