## Methodologie HearAdvisor

- **SoundGrade** : A-F (A = meilleur)
- **SoundScore** : 0-5 (score numerique)
- **Tests** : 72 scenes acoustiques, laboratoire GRAS, array 8 haut-parleurs
- **2 conditions** : Initial Fit (reglage usine) vs Tuned Fit (reglage audioprothesiste optimal)
- **5 sous-scores** : Speech in Quiet, Speech in Noise, Feedback Handling, Own Voice Comfort, Music Streaming
- **Perte ciblee** : Moderee sensorineurale
- **Reviewers** : Dr. Steve Taddei Au.D., Dr. Abram Bailey Au.D.

## Insights cles pour notre grille de notation

1. **Speech-in-noise est LE differenciateur** — ecart de 0.4 a 5.0, c'est la ou les produits se separent vraiment
2. **Feedback handling = presque tous bons** (4.0-5.0 initial) mais DROP severe apres tuning pour beaucoup
3. **Own-voice comfort** — tres variable (2.1 a 5.0), correle au type de dome/coupling
4. **Musique streaming** — point faible generalise, rare au-dessus de 4.0
5. **Initial vs Tuned** — certains produits s'ameliorent massivement (Phonak +2.1, Starkey Genesis +1.33), d'autres peu (ReSound +0.06)
6. **Prix vs performance** — Rexton Costco = meilleur rapport qualite/prix, ReSound = pire rapport
7. **ReSound = faiblesse bruit** — les 2 produits scores < 1.5 en speech-in-noise
